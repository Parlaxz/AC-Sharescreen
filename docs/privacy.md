# Privacy

## Data Collected

- Host display name (user-provided)
- Share/viewer tokens (random, non-identifying)
- Session timestamps
- Display metadata (resolution, count)
- Connection statistics (bitrate, packet loss, RTT)

## Not Collected

- Any content from the shared screen
- Audio content
- Personal identifying information beyond display name
- IP addresses in logs (redacted)
- Full peer UUIDs in logs (shortened)

## Viewer Link

The permanent viewer link contains only random tokens.
It does not contain the host's identity, location, or VDO credentials.
